---

## 📱 概要
「ネトたま」は、GIGAスクール構想下で1人1台端末を持つ小学生を対象とした、**情報モラル教育用iOSアプリケーション**です。SNSの適切な利用方法やネットリテラシーを、クイズやシミュレーションを通じて直感的に学習できる環境を提供します。

## 💡 開発背景
インターネット利用の低年齢化が進む中、従来の座学中心の指導では「自分事」として捉えにくいという課題がありました。「実体験に近い形で、楽しく学べる教材を」という教育現場のニーズに応えるため、研究プロジェクトとして開発をスタートしました。

## ✨ 主な機能
- **シナリオ学習**: ネット上でのトラブル事例を擬似体験
- **モラルクイズ**: 実践知識に合わせた知識定着
- **直感的なUI**: 児童が迷わず操作できる、文字を最小限にしたデザイン

## 🛠 使用技術
- **言語**: Swift
- **フレームワーク**: SwiftUI
- **開発環境**: Xcode

## 🚀 技術的・工夫したポイント
- **ユーザー（児童）への徹底した寄り添い**
  何度も教育センターと協議を重ね、フィードバックを収集。ボタンの配置や色の識別性、言葉の選び方をアジャイルに改善し続けました。
- **表現力の追求**
  学習のハードルを下げるため、アニメーションを導入し、ゲームのような没入感のあるUXを実現しました。
